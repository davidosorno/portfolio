SALES APP

This application was created to help the company grow and improve the sales system.
It allows to store general information about each customer, also the products that they buy, average amount, purchase history, print the invoice by bluetooth and after saving the information, the application sends it to the server.
It also allows to have different user roles such as general manager, sales manager, seller, and so on, and handle inventory of the products that the seller still has.

REGULATIONS AND RAW MATERIAL

System created to monitor government compliance for health and safety code regulations. In addition, the application handles inventory of raw material, entry and exit of workers, and some formats for Invima (The Colombia National Food and Drug Surveillance Institute).

DESKTOP APP

App to manage datas saved by both sellers and production manager in the Android Apps that we have connected to same database.
It allows to see the bills that we have not paid of raw material and check the docuemnts that we must have in order when the Invima requests them.
Sales report, create or delete employes, see invoice saved by the sellers, trusted by the seller, average sales are some app's features.

SINGLE PACMAN

Single and Multiplayer Pacman were made as a code practice in Coding Dojo.

MULTIPLAYER PACMAN

Single and Multiplayer Pacman were made as a code practice in Coding Dojo.
To develop the multiplayer Pacman, I used the Node.js with built-in Socket technology that allows direct communication between connected systems.
Like the Single Pacman, the world is random and change every time the Game start, but in this case, the world change for every player too, and all the players can see the progress for the other.
